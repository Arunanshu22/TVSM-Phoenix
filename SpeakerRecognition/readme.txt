Speaker recognition with 5 speakers using mfcc.
The model "Five_speaker_model_best1.joblib" gave training accuracy of 99.58% & testing accuracy of 99.19%. Its code is written in the file mfccfivespeaker5.ipynb. 
The model "Five_speaker_model_4rd_best.joblib" gave training accuracy of 98.68% & testing accuracy of 87.06%. Its code is written in the file mfccfivespeaker9.ipynb.

Voice_recognition_speech_brain.ipynb - It perform speaker verification with a pretrained ECAPA-TDNN model using SpeechBrain. The system can be used to extract speaker embeddings as well. It is trained on Voxceleb 1+ Voxceleb2 training data.
The link is given below - https://huggingface.co/speechbrain/spkrec-ecapa-voxceleb
